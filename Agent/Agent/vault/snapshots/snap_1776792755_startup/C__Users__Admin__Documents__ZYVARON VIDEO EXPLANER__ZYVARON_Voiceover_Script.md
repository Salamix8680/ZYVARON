# ZYVARON — Explainer Voiceover Script
### For screen recording with OBS / Windows Game Bar
**Total runtime: ~75 seconds | 9 slides**

---

## HOW TO RECORD

1. Open `ZYVARON_Explainer.html` in Chrome — press `F11` for fullscreen
2. Press `Win + G` → click **Start Recording** (or use OBS)
3. Read each script line as the slide appears
4. The slides auto-advance — follow the timing below
5. Use a deep, slow, confident voice — pause on the `...` marks

## PRONUNCIATION GUIDE

| Written | Say it like |
|---------|-------------|
| ZYVARON | **ZI-vah-ron** |
| ZI | like the word "eye" with a Z — same as "xylophone" starts |
| VAH | like "vah" in "java" |
| RON | like the name "Ron" |

**Memory trick:** Think "ZI-vah-ron" = *"Zy like rye, vah like spa, ron like con"*
